<SendTele>
기능 : 우분투 터미널에서 텔레그램봇 메시지 전송

<사용 방법>
00.텔레그램 봇 생성, 토큰, 챗아이디 얻기(인터넷 검색)
   
01.텔레그램 파이썬 모듈 설치
   pip3 install python-telegram-bot
   
02.텔레그램 토큰, 챗아이디를 sendtele.ini 파일에 입력
  
03.우분투 터미널에서 sendtele.ini, sendtele.py 파일이 있는 경로로 이동
   sendtele.py "보내고 싶은 메시지"
